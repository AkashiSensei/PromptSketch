# RAW_REQUIREMENTS

记录时间：2026-07-07

## 产品意图

PromptSketch 是一个我自己用的轻量草图工具，主要用于快速给 AI 工具画“图形提示词”。我希望能用它快速表达布局、组件位置、UI 想法和粗略草图，而不是用很复杂的文字描述组件应该怎么摆放。

这个工具不是专业绘画软件，也不是白板或设计软件。它更像一个“视觉想法中转站”：把截图或空白画布打开，随手画几笔，然后快速复制到剪贴板或保存成图片。

## 核心需求

- 极致轻量，打开很快，上手很快。
- 可以作为网页访问，并部署到 GitHub Pages。
- 可以把当前剪贴板里的图片直接粘贴到画布上，通常是截图或参考图。
- 支持用鼠标、触摸板、数位板或触控笔快速画画。
- 在设备和浏览器支持的情况下，画笔要支持压感。
- 能够非常快地把当前合成后的画布复制到系统剪贴板。
- 需要时可以把当前合成后的画布保存为本地图片。
- 要有简单可靠的橡皮。
- 橡皮应该只擦掉我画的标注，不要破坏粘贴进来的截图、底图或背景色。
- 必须支持撤销功能，这一点很关键。
- 支持简单形状：矩形、圆形 / 椭圆、圆角矩形，可能还需要直线。
- 颜色能力应从画笔设置中独立出来，由画笔和形状共享同一个当前颜色；形状只在线框和纯色实心两种模式间切换，实心模式不绘制线框。
- 颜色选择固定在控制面板上方，切换到橡皮时保持原位并显示为禁用状态，避免工具选项变化造成定位跳动。
- 允许用户设置快捷键。
- 允许用户设置画笔颜色、粗细、透明度等简单画笔选项。
- 不打算做项目文件、文档管理或作品保存。
- 可以保存用户偏好设置，例如快捷键、画笔颜色、画笔粗细，但不需要保存绘图作品本身。

## 期望交互

- 打开页面后可以立刻开始画。
- `Cmd+V` / `Ctrl+V` 可以把剪贴板图片作为当前画布的底图或基础图像。
- `Cmd+C` / `Ctrl+C` 可以把当前合成结果作为图片复制到剪贴板。
- `Cmd+Z` / `Ctrl+Z` 应该可以撤销最近的绘制或编辑操作。
- 橡皮在黑底画布、白底画布和截图上都应该表现一致，不要出现把背景擦成白色之类的问题。
- UI 要保持极简，不要有完整绘画软件或复杂设计工具的负担。

## 未来绘制体验想法

- 手绘线条要尽量平滑，减少抖动。
- 压感可以影响线条粗细或透明度。
- 可以适当优化手绘线条。
- 如果画一条比较长、接近直线的线，并且末尾不抬笔停顿一小段时间，工具可以自动把它优化或预览成直线。
- 后续也可以考虑曲线拟合或简单形状识别，但这不是第一版最核心的内容。
